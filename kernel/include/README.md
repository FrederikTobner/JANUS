# include — Global Headers

Global, cross-cutting header files used throughout JANUS.

## Purpose

Provides kernel-wide type definitions, attributes, configuration, and version
information accessible to all modules.

## Structure

```text
include/
├── janus/               # Main OS interface
│   ├── types.h          # Global kernel types (u8–u64, phys_addr_t, virt_addr_t, pid_t, etc.)
│   ├── attributes.h     # Compiler attributes (__noreturn, __packed, etc.)
│   └── config.h.in      # Build configuration template (processed by CMake)
└── uapi/                # User-space API headers (future)
```

Note: `config.h` is generated by CMake in the build directory (`build/include/janus/config.h`).

## Usage

```c
#include <janus/types.h>

phys_addr_t phys_addr = 0x100000;
virt_addr_t virt_addr = 0xFFFFFFFF80000000;

#include <janus/attributes.h>

__noreturn void kernel_main(void);
```
