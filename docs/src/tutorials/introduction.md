# JANUS Tutorials

These tutorials are hands-on, step-by-step guides that walk through building something from scratch. Each one isolates a single concept and explains it in concrete terms — the reader writes real code and sees real results.

Think of them as the practical counterpart to the [Concepts](../concepts/introduction.md) section. Where a wiki article explains *what* a bump allocator is and *why* it works, the corresponding tutorial here walks through *writing one*, line by line.

Not every concept has a tutorial. A guide only appears here when the topic genuinely benefits from a guided walkthrough and can be explained in isolation, without requiring the full JANUS codebase to follow along.

Tutorials use `mdbook-code-diff` annotations where appropriate, so that complex functions can be examined one line at a time rather than presented as a wall of code with interleaved commentary.
