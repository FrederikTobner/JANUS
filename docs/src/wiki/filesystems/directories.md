# Directories

Directories are used to organize files and other directories. 
They can be nested within each other to create a hierarchical structure.
Similar to files, directories have names and permissions, but they do not contain data in the same way that files do.
Directories can contain files, other directories, which are called subdirectories, and links to files or other directories.
Because links can point to files or  directoriries from a different level in the hierarchy, file systems are therefor considered to be directed acyclic graphs (DAGs) rather than strict trees.
