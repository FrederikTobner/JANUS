# JANUS Documentation

JANUS (Just ANother Unix-like System) is an educational operating system kernel written in C17 and assembly, targeting x86_64 and aarch64. This documentation is organised into four parts:

- **Wiki** — general OS and computer science topics: hardware interfaces, boot sequences, memory management, and developer tooling. The articles are not specific to JANUS; they aim to explain the underlying theory clearly enough for anyone with a reasonable grasp of C.
- **Tutorials** — hands-on, step-by-step guides that walk through building something from scratch. Each tutorial isolates a single concept and makes it concrete.
- **Reference** — JANUS-specific architecture, subsystem design, and the reasoning behind key design decisions. This section deliberately stays at the architectural level; function-level documentation lives in the source as Doxygen comments.
- **Project** — coding style, naming conventions, technical philosophy, and build setup for contributors.
