# JANUS Documentation

Documentation of the JANUS project, seperated into three different sections:

- **Wiki** — general OS and computer science topics: hardware interfaces, boot sequences, memory management, I/O, and graphics. The articles are not specific to JANUS; they aim to explain the underlying theory clearly enough for anyone with a reasonable grasp of C.
- **Tutorials** — hands-on, step-by-step guides that walk through building something from scratch. Each tutorial isolates a single concept and makes it concrete. Tutorials are grouped by domain: boot, I/O, graphics, and debugging.
- **Project** — JANUS-specific architecture, subsystem design, and the reasoning behind key design decisions. Also covers coding style, naming conventions, technical philosophy, developer tooling, and build setup for contributors.
