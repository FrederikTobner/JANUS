# CI

All workflow files live in `.github/workflows/`.

## build.yaml — Kernel build matrix

Triggers on pull requests targeting `main`.

Builds the kernel across all four preset configurations:

| Preset | Arch | Compiler |
|---|---|---|
| `x86_64-gcc` | x86_64 | GCC |
| `x86_64-clang` | x86_64 | Clang |
| `aarch64-gcc` | aarch64 | GCC |
| `aarch64-clang` | aarch64 | Clang |

Each cell runs: **Configure → Build → Build ISOs**.

The `x86_64-clang` cell additionally runs:

- **clang-format** — `clang-format --dry-run --Werror` over every `kernel/**/*.{c,h}`. Fails if any file diverges from `.clang-format`.
- **clang-tidy** — static analysis against the generated `compile_commands.json`. Errors abort the job.

## tools.yaml — Tools build, test, and coverage

Triggers on pull requests and pushes to `main` when `tools/**` changes.

- **build-and-test:** builds and runs CTest for the tools sub-project with GCC and Clang.
- **coverage:** builds with GCC + gcov instrumentation, runs all tests, produces an HTML coverage report uploaded as artifact `coverage-report` (30-day retention).

Run coverage locally:

```bash
cmake -S tools -B build-tools-coverage \
    -G Ninja \
    -D CMAKE_BUILD_TYPE=Debug \
    -D CMAKE_C_COMPILER=gcc \
    -D TOOLS_ENABLE_COVERAGE=ON
cmake --build build-tools-coverage
ctest --test-dir build-tools-coverage --output-on-failure
gcovr --root tools --object-directory build-tools-coverage \
      --html-details coverage/index.html --print-summary
```

## doxygen.yaml — API documentation

Triggers on pull requests and pushes to `main` when `kernel/**`, `tools/**`,
or `Doxyfile` changes.

- **build:** generates Doxygen HTML, uploads `doxygen-html` artifact (30-day retention). Runs on every push and PR.
- **publish** (push to `main` only): deploys Doxygen HTML directly to GitHub Pages at `/JANUS/`.

## Quality Gate Summary

| Check | Workflow | When |
|---|---|---|
| Kernel builds (all 4 presets) | `build.yaml` | Every PR |
| `clang-format` | `build.yaml` | Every PR |
| `clang-tidy` | `build.yaml` | Every PR |
| Tools build + CTest | `tools.yaml` | Every PR touching `tools/` |
| Tools coverage report | `tools.yaml` | Every PR touching `tools/` |
| Doxygen validation | `doxygen.yaml` | Every PR touching `kernel/tools/Doxyfile` |
| Pages deploy | `doxygen.yaml` | Push to `main` |
