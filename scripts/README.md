# TinyOS Build and Utility Scripts

This directory contains shell scripts for building, testing, and running TinyOS.

## Planned Scripts

- **build.sh** - Automated build script with common configurations
- **run-qemu.sh** - Launch TinyOS in QEMU for testing
- **debug-qemu.sh** - Launch TinyOS in QEMU with debugging enabled
- **gen-iso.sh** - Generate bootable ISO image
- **clean.sh** - Clean build artifacts

## Usage

Scripts will be added as the project develops. Each script will include usage documentation in its header comments.
